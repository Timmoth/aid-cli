aid disk info

  aid disk info  Show disk information
            -j, --json  Output disk information in JSON format.

-----input-----
aid  disk info -j
-----output-----
[
  {
    "available_gb": 52.2,
    "total_gb": 936.2,
    "percent_free": 5.6,
    "file_system": "NTFS",
    "mount_point": "C:\\",
    "removable": false,
    "kind": "SSD"
  }
]
